# 🌟 GlassNote - Advanced Aesthetic Note App
A modern, full-featured Android note application with glassmorphism UI, auto-save, local storage, and advanced file handling. Built to run & compile **entirely inside GitHub Codespaces** via a single bash script.

## 🚀 Quick Start in GitHub Codespaces
1. Create a new Codespace (Ubuntu/Java/Android base image recommended).
2. Upload or paste `setup.sh` into the workspace root.
3. Make it executable & run:
   ```bash
   chmod +x setup.sh && ./setup.sh
